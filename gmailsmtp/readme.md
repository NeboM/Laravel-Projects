# GMAIL SMTP

Send emails using gmail with Laravel

Clone the project and edit your .env file

> MAIL_DRIVER=smtp

> MAIL_HOST=smtp.googlemail.com

> MAIL_PORT=465

> MAIL_USERNAME=YOUR_GMAIL_EMAIL

> MAIL_PASSWORD=YOUR_GMAIL_PASSWORD

> MAIL_ENCRYPTION=ssl
